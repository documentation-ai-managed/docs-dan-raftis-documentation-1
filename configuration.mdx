## Environment Variables

Use environment variables for sensitive configuration and deployment-specific settings.

| Variable | Description | Example |
|----------|-------------|---------|
| `DATABASE_URL` | Database connection string | `postgresql://user:pass@host/db` |
| `API_SECRET` | API authentication secret | `sk-1234567890abcdef` |
| `REDIS_URL` | Redis cache connection | `redis://localhost:6379` |

<Expandable title="Environment Setup" default-open="true">
  <Steps>
    <Step title="Create .env File" icon="file">
      Create a `.env` file in your project root.
    </Step>
    <Step title="Set Variables" icon="settings">

    </Step>
    <Step title="Load Variables" icon="download">

</Expandable>
